# Sales-data-dashboard
1.Headline

creative interactive dashboard to track and analyze for online sales data 
A dynamic, Interactive Data visualization tool build to explore the online sales data-focusing on profit,quantity,amount,category and their sum etc.

2.Short Description

Used complex parameter to drill down in worksheet and customization Using filter and slicer. Create connection, joint new table, calculation to manipulate data and enable user driven parameter for visualization.

3.Customised visualization

Use different type of customized visualization like bar graph pie chart, donut chart,clustered bar graph,scattered chart,line chart, area chart, map, slicer etc.

4.Data Source

From youtube video of (Power Bi tutorial for Beginners | Power BI Dashboard Project in Hindi | Power Bi Course) from Rishabh Mishra
